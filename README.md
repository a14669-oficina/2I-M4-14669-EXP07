<h2>🎯 Objetivo do Projeto 🎯</h2>
O objetivo deste projeto foi desenvolver uma página web mais avançada com recurso a PHP e formulários, permitindo não só a introdução de dados pelo utilizador, mas também o seu processamento no servidor. Este tipo de sistema simula aplicações reais, como formulários de registo ou recolha de informação que depois pode ser enviada ou armazenada numa base de dados.

<h2>🗂️ Estrutura do Projeto 🗂️</h2>
O projeto encontra-se organizado numa estrutura simples de páginas web, sendo a principal um formulário HTML ligado a ficheiros PHP responsáveis pelo processamento dos dados. O sistema está alojado online através de um servidor gratuito, permitindo o acesso via browser.

A estrutura inclui:

Página HTML com formulário
Ficheiro PHP para tratamento dos dados enviados
Campos de input para recolha de informação
Botão de submissão do formulário

Esta organização permite separar a parte visual (frontend) da parte lógica (backend), o que é fundamental no desenvolvimento web.

<h1>⚙️ Funcionalidades Desenvolvidas ⚙️</h2>
O sistema desenvolvido apresenta várias funcionalidades importantes:

Recolha de dados através de formulário HTML
Envio de dados via método POST
Processamento dos dados com PHP
Ligação entre frontend e backend
Possibilidade de validação e tratamento da informação enviada

O formulário permite ao utilizador introduzir dados e enviá-los para serem processados pelo servidor, demonstrando o funcionamento básico de uma aplicação web dinâmica.
